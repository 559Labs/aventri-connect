JsonMapper

class JsonMapper

Automatically map JSON structures into objects.

Properties

protected object $logger PSR-3 compatible logger object
bool $bExceptionOnUndefinedProperty Throw an exception when JSON data contain a property that is not defined in the PHP class
bool $bExceptionOnMissingData Throw an exception if the JSON data miss a property that is marked with @required in the PHP class
bool $bEnforceMapType If the types of map() parameters shall be checked.
bool $bStrictObjectTypeChecking Throw an exception when an object is expected but the JSON contains a non-object type.
bool $bStrictNullTypes Throw an exception, if null value is found but the type of attribute does not allow nulls.
bool $bIgnoreVisibility Allow mapping of private and proteted properties.
bool $bRemoveUndefinedAttributes Remove attributes that were not passed in JSON, to avoid confusion between them and NULL values.
array $classMap Override class names that JsonMapper uses to create objects.
callable $undefinedPropertyHandler Callback used when an undefined property is found.
protected array $arInspectedClasses Runtime cache for inspected classes. This is particularly effective if mapArray() is called with a large number of objects
string|null $postMappingMethod Method to call on each object after deserialization is done.
