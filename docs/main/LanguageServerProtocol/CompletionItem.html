CompletionItem

class CompletionItem

Properties

string $label The label of this completion item. By default also the text that is inserted when selecting this completion.
int|null $kind The kind of this completion item. Based of the kind an icon is chosen by the editor.
string|null $detail A human-readable string with additional information about this item, like type or symbol information.
string|null $documentation A human-readable string that represents a doc-comment.
string|null $sortText A string that should be used when comparing this item with other items. When falsy the label is used.
string|null $filterText A string that should be used when filtering a set of completion items. When falsy the label is used.
string|null $insertText A string that should be inserted a document when selecting this completion. When falsy the label is used.
TextEdit|null $textEdit An edit which is applied to a document when selecting this completion. When an edit is provided the value of insertText is ignored.
TextEdit[]|null $additionalTextEdits An optional array of additional text edits that are applied when selecting this completion. Edits must not overlap with the main edit nor with themselves.
Command|null $command An optional command that is executed after inserting this completion. Note that additional modifications to the current document should be described with the additionalTextEdits-property.
mixed $data An data entry field that is preserved on a completion item between a completion and a completion resolve request.
int|null $insertTextFormat The format of the insert text. The format applies to both the insertText property and the newText property of a provided textEdit.
