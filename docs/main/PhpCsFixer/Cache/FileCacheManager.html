FileCacheManager

final class FileCacheManager implements CacheManagerInterface

Class supports caching information about state of fixing files.

Cache is supported only for phar version and version installed via composer.

File will be processed by PHP CS Fixer only if any of the following conditions is fulfilled: - cache is corrupt - fixer version changed - rules changed - file is new - file changed
