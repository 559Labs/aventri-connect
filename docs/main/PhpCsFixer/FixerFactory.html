FixerFactory

final class FixerFactory

Class provides a way to create a group of fixers.

Fixers may be registered (made the factory aware of them) by registering a custom fixer and default, built in fixers. Then, one can attach Config instance to fixer instances.

Finally factory creates a ready to use group of fixers.
